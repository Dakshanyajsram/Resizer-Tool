# Resizer-Tool
# 🖼 Image Resizer Tool

## 📌 Objective
Resize and convert multiple images in a folder using Python and Pillow.

## 🚀 Features
- Batch image resizing
- Converts image formats (JPG to PNG etc.)
- Automatically creates output folder
- Handles errors using try-except
- Simple and beginner-friendly

## 🛠 Technologies Used
- Python 3
- Pillow (PIL)
- OS module

## 📂 How It Works
1. Reads all images from `input_images` folder.
2. Resizes them to specified width and height.
3. Converts them to selected format.
4. Saves them in `output_images` folder.
